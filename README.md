## 개발 환경 설정

### 필수 요구사항

- Node.js 18+
- npm 또는 yarn

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 코드 포매팅
npm run format

# 타입 검사
npm run type:check

# 테스트 실행
npm run test:unit
npm run test:e2e
```
